# Pomodoro Timer

A simple and elegant Pomodoro Timer web application built with HTML, CSS, and JavaScript. This application helps you stay focused and productive using the Pomodoro Technique.

## Features

- Clean and intuitive user interface
- Toggle between Work and Break modes
- 25-minute work sessions (customizable)
- Start and Reset functionality
- Visual mode indicator
- Responsive design

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ChristoBD/BYOA-Pomodoro.git
```

2. Navigate to the project directory:
```bash
cd BYOA-Pomodoro
```

3. Open `index.html` in your web browser

## Usage

1. Click the "Start" button to begin a Pomodoro session
2. The timer will count down from 25 minutes
3. Use the toggle switch to switch between Work and Break modes
4. Click "Reset" to reset the timer to 25:00
